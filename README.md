# Super Hack Installer Beta

Super Hack Installer Beta là một **installer** dành cho Geometry Dash, giúp bạn cài đặt hack/extension một cách dễ dàng.  
⚠️ Đây **không phải** là mod menu.

## ✨ Tính năng
- Popup nhập license key để xác thực.
- Nút "Install" trực quan, dễ sử dụng.
- Kiểm tra key hợp lệ trước khi cài đặt.
- Giao diện đơn giản, dễ hiểu.

## 📂 Cấu trúc
- `src/main.cpp` : code chính, xử lý UI và logic nhập key.
- `build/` : thư mục chứa file build (DLL, .geode).
- `README.md` : tài liệu mô tả dự án.

## 🚀 Cách build
1. Clone repo:
   ```bash
   git clone https://github.com/Bil966/Super-Hack-Installer-Beta.git
2. Vào thư mục dự án:
   cd Super-Hack-Installer-Beta
3. Build bằng CMake:
   geode build
## 🛠️ Yêu cầu
Visual Studio + CMake

Geode SDK

Geometry Dash (để test plugin)
